1. Introduction

Do you ever felt overwhelmed by the on going constructions near you? Do you want to know what is being built? Do you wish to check out the documents and papers as well as use your right to influence the urban projects?

The NYC Urban Projects Map helps you check them out and participate in the NYC’s Urban Development Process easily!

3. NYU Capstone Project

3.1 Capstone Brief

This is a NYU capstone project with sponsorship from InCitu. We were helping InCitu to develop their database by collecting more data from various NYC data portals. The ultimate task is use those data to build a map. During the project, we collected and reformated two data tables and automated the process. In the end we successfully developed the map based on those data.

3.2 Data Automation Github Repo

This GitHub Repository is mainly for data collection automation pipeline. It uses GitHub Actions to scrape the data sources and integrate them together.

Repo: https://github.com/ZephyrCW/cusp-capstone
